# Lake Apps Website by Deividas Strole

A modern static website for **Lake Apps**, created by **Deividas Strole**.
The website is built with simple front-end technologies and designed for fast deployment on **GitHub Pages**.

## Overview

Lake Apps is a digital brand focused on modern websites, custom software, and digital growth solutions. This static website presents the company’s services, team structure, and contact information in a clean, professional, and responsive design.

The project uses a minimalist black-and-white visual style with electric gradient accents inspired by the following color palette:

```css
linear-gradient(135deg, #ff006e, #2563eb, #ffbe0b)
```

## Features

* Static website with no build tools required
* Home, About, and Contact sections
* Responsive layout for desktop and mobile screens
* Modern black-and-white design
* Gradient branding elements
* Animated sections and hover effects
* SVG illustrations
* Scroll-based rotating team images
* Simple front-end contact form placeholder
* Ready for GitHub Pages deployment

## Technologies Used

* HTML5
* CSS3
* JavaScript
* SVG graphics
* GitHub Pages

## Pages

### Home

The Home section introduces Lake Apps and highlights the company’s main services:

* Websites
* Web apps
* Digital growth

### About

The About section explains the company’s approach and includes an “Our Team” section with animated team images.

### Contact

The Contact section includes a simple front-end form layout. The form is currently a demo and can later be connected to a backend service, Formspree, Netlify Forms, or another email handling solution.
